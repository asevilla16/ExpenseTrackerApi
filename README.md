# Run in development

1. Clone Repository
2. Run

```
npm install
```

3. Must have NestJS installed

```
npm i -g @nestjs/cli
```

4. Set up database

```
docker-compose up -d
```

5. Run the project

```
npm run start:dev
```

6. Run tests

```
npm run test:watch

npm run test:cov
```

## Stack

- MongoDB
- Nest
